## 06_if_elif_else_mood_checker

**Python Conditional Statements (if / elif / else)**

This program demonstrates how to use conditional logic to respond differently based on user input.

---

##  What This Program Does

The program:

1. Asks the user about their current mood.
2. Stores the input in a variable called `mood`.
3. Uses `if`, `elif`, and `else` statements to check the value.
4. Prints a different message depending on the mood.

---

# 💻 Python Code

```python
# Ask the user about their mood
mood = input("What is your mood? (happy/sad/other): ")

# Check the mood
if mood == "happy":
    print("Keep smiling!")
elif mood == "sad":
    print("Don’t worry, be happy!")
else:
    print("Have a good day anyway!")
```
