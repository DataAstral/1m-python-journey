## ➤ loops

This folder contains my Python practice projects focused on loops and repetition.

Here I learned how programs:

- repeat actions with `while` and `for`  
- generate sequences with `range()`  
- stop or skip loop steps using `break` and `continue`  
- work with input inside loops  
- count, sum, and search through data  
- build patterns with nested loops
