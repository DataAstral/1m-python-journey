# **👤** d05_data_types_user_profile

## ⚙️ Topic

**Python Variables & Data Types**

This program demonstrates how to store user information using different data types and display both values and their types.

---

# 🧠 What This Program Does

The program:

1. Creates variables for a user profile (name, age, city, height, student status).
2. Uses correct Python data types: `str`, `int`, `str`, `float`, `bool`.
3. Prints all values to the screen.
4. Prints the type of each variable using `type()`.

---

# 💻 Python Code

```python
# Create user profile variables
name = "Anna"          # string
age = 25               # integer
city = "London"        # string
height = 1.68          # float
is_student = True      # boolean

# Print values
print(name)
print(age)
print(city)
print(height)
print(is_student)

# Print types (debugging / learning purpose)
print(type(name))        # name
print(type(age))         # age
print(type(city))        # city
print(type(height))      # height
print(type(is_student))  # is_student
```

---

# 🧩 Notes

- Each variable has a specific data type:
  - `str` → text
  - `int` → whole numbers
  - `float` → decimal numbers
  - `bool` → True / False
- `type()` helps you understand what kind of data you are working with.
- This is a basic foundation for building real applications (forms, databases, APIs).
